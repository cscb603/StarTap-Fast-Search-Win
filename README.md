## 🌟 为什么要在你的电脑里放一个“星TAP”？

如果你觉得 Windows 自带的搜索太慢，而原版的 Everything 界面又太像“古董表格”，那么星TAP 就是为你准备的。

### 1. 看着顺眼，用着舒心
*   **字大清晰**：专为高分屏优化，字体大而圆润，不用眯着眼找文件。
*   **清爽界面**：采用高级的莫兰迪配色，不管是白天的经典蓝白还是晚上的深色模式，都干净利落，不伤眼。
*   **沉浸式体验**：没有复杂的菜单和密密麻麻的按钮，只有你需要的搜索框和结果。

### 2. 分类简单，一步到位
*   **专治“找不着”**：想找刚下载的文档？想找昨天的视频？点击顶部的“文档”、“图片”或“视频”分类，瞬间筛选，不用记任何复杂的搜索指令。
*   **桌面/程序优先**：我们把“桌面”和“软件目录”放在了最高优先级。想开个软件、找个桌面文件，输入名字首字母就出来了。

### 3. 步骤最少，速度最快
*   **搜完即走**：点击托盘图标弹出，搜到文件回车打开，窗口自动隐藏，绝不占用你的桌面空间。
*   **别名搜索**：输入 `ps` 就能找到 Photoshop，输入 `浏览器` 就能找到 Chrome。我们帮你记住了那些复杂的英文名。
*   **右键复制路径**：搜到结果后，右键一点即可“复制完整路径”，再也不用手动去属性里翻地址了。

### 4. 越用越懂你的“智能补全”
*   **自我学习**：星TAP 会悄悄记住你的搜索习惯。你经常点的文件，下次只需输入前几个字母，它就会作为“智能建议”浮现在搜索框下方，一步直达。
*   **非侵入提示**：这些建议只是轻巧的提示，不会强行干扰你的打字流程，既贴心又自然。

### 5. 绿色纯净，安全放心
*   **解压即用**：不需要安装，不写注册表，不喜欢了直接删掉文件夹就行。
*   **绝无冲突**：我们移除了所有可能导致冲突的热键，它会安静地待在托盘里，随叫随到。
*   **不需要安装 Everything！**：完整的 Everything 核心组件已经打包在 `lib` 文件夹里了，直接运行即可！

### 6. 智能时间主题
*   **自动切换**：根据时间自动选择主题
  - 6:00 - 18:00 → 白昼模式（经典蓝白）
  - 18:00 - 6:00 → 暗夜模式（深色）
*   **可手动覆盖**：右上角图标按钮随时切换

### 7. 输入法说明 ⚠️
*   **输入英文**：请切换到英文输入法
*   **输入中文**：请切换到中文输入法
*   **注意**：当前版本暂不支持中英文混合输入（需要分开切换输入法）

## 📖 怎么开始使用？

### 1. 快速上手
1.  双击运行文件夹里的 `星TAP极速搜索.exe`。
2.  在屏幕右下角的任务栏里找到“小星星”图标，点它一下。
3.  **换个心情 (切换颜色)**：在搜索框的最右侧，有一个精美的图标按钮，点一下即可在“经典蓝白”和“暗夜模式”之间一键切换。
4.  直接打字，结果就出来了！

### 2. 目录结构说明 (绿色版)
解压后的文件夹结构如下：
```text
dist/
├── 星TAP极速搜索.exe          # 主程序（双击运行）
├── README.md                  # 你正在阅读的说明书
├── ARCHITECTURE.md            # 技术架构与进阶说明
├── assets/                    # 资源文件夹
│   └── ai搜索.ico             # 程序图标
├── lib/                       # 核心依赖库（不可删除！）
│   ├── es.exe                 # Everything 命令行工具
│   ├── Everything.exe         # Everything 引擎程序
│   └── Everything64.dll       # 引擎动态链接库
├── 极速搜索 界面图 白天1.png  # 白昼模式截图
└── 极速搜索 界面图 黑夜1.png  # 暗夜模式截图
```

### 3. 文件功能详解
- **星TAP极速搜索.exe**：采用 Rust 编写的现代化前端，负责所有 UI 显示、托盘管理和用户交互。
- **lib/** 文件夹：**千万不要删除！** 这里面包含了完整的 Everything 搜索引擎，程序启动时会自动加载。

### 快捷操作
- **呼出/隐藏**: `Ctrl + Shift + F` (全局热键)
- **选择**: `↑` / `↓` 方向键
- **打开**: `Enter` 回车键
- **退出**: `Esc` 键 或 托盘菜单
- **复制路径**: 结果项右键菜单
- **主题切换**: 顶部图标按钮

### 核心特性
- **极速启动**: 采用 Rust 开发，内存占用极低，毫秒级搜索响应。
- **智能补全**: 基于您的点击频率自动学习，高频文件搜索更快捷。
- **外观精美**: 莫兰迪色系 UI，支持深色/浅色模式切换，完美支持高分屏。
- **绿色免安装**: 纯净无后台，Everything 引擎已内置，零配置开箱即用。
- **智能时间主题**: 6:00-18:00 自动白昼模式，其他时间暗夜模式。
- **中英文输入**: 支持中文和英文输入（需切换对应输入法）。

### 常见问题
- **如何使用 AI 搜索？** 
  AI 增强功能正在内测中，当前版本专注于提供极致的本地文件搜索体验。
- **为什么搜不到文件？** 
  请确保 `lib` 文件夹存在且完整，程序会自动启动 Everything 服务。
- **系统支持**: 完美支持 Windows 10 / 11。
- **输入法问题**: 输入英文请切英文输入法，输入中文请切中文输入法。
